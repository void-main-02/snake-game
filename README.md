# 🐍 Snake Game

A classic Snake Game built using basic web technologies. Control the snake, eat food, grow longer, and try to beat your high score!

---

## 🎮 Features

- Smooth snake movement
- Score tracking system
- High score saving
- Game over detection
- Simple and clean UI
- Keyboard controls

---

## 🛠️ Technologies Used

- **HTML** – Structure of the game  
- **CSS** – Styling and layout  
- **JavaScript** – Game logic and interactivity
